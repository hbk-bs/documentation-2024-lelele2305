In dieser Dokumentation wird sich mit dem Lernstoff des letzten Semesters Grundlagen der digitalen Kommunikation auseinandergesetzt. Neben den einzelnen Themen wurde sich im Zusammenhang dazu, intensiver mit der Programmierung und Darstellung von Websiten beschäftigt. Sowohl HTML, als auch CSS und JavaScript kamen hier zum Einzatz, um bestimmte Elemente einzupflegen und zu veranschaulichen bzw. dessen Funktionen zu definieren. Durch die Einführung in diese Programmiersprachen im vorherigen Grundlagenseminar konnte erfolgreich auf diesem Wissen aufgebaut werden.

Zu Beginn wird das erste Thema der sogenannten Teachable Machine untersucht. Wie der Name schon verrät, handelt es sich hierbei darum, einer machine Wissen anzulernen und dadurch Modelle für machinelles Lernen zu erschaffen, die es einem ermöglichen, verschiedene Aufgaben wie Bilderkennung, Sounderkennung oder Personenabschätzung durchzuführen. Hierzu wird das ausgearbeitete Modell „Plant Pests AI” vorgetragen und erläutert. Dieses ist ein Bilderkennungsmodell.

Als nächstes wird das Large Language Model (LLM) betrachtet. Es ist ein maschinelles Lernmodell, welches darauf trainiert ist, Sprache zu verstehen und zu generieren. Auf ein riesigen Datenmenge und komplexen neuronalen Netzwerken basiert es. Um dies genauer zu verstehen, wird das LLM namens Ollama vorgestellt, welches seit April 2024 auf dem Markt ist. Auch hier wird anschließend ein Beispiel aufgeführt. Der „Fairytale Generator” basiert auf einem LLM.

Das Kapitel Generative Geschichte basiert auf dem Können von künstlicher Intelligenz in Schrift. Es ist eine Geschichte, die von einem Algorithmus oder Modell erstellt wird. Sie basiert auf Mustern, anstatt von Mneschen geschrieben zu werden. Diese Muster hat das Modell aus großen Mengen von Textdaten gelernt. Mit Hilfe von ChatGPT kann eine solche Geschichte geschrieben werden. Im fogenden Beispiel, „Pforten des Universums”, wird demonstriert, was solch eine künstliche Intelligenz kann und was möglich ist.
