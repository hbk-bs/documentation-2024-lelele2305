Was ist eine Teachable Machine?

Eine Teachable Machine ist ein benutzerfreundliches Tool von Google. Es ermöglicht, eigene machinelle Lernmodelle zu erstellen, ohne die Voraussetzung für ausgeprägte Programmierkenntnisse. Es ist darauf ausgelegt, die Grundlagen des machinellen Lernens zugänglich und verständlich zu machen.

Durch die einfache Bedienung können Nutzer Modelle erstellen, indem sie Daten hochladen oder einfach aufnehmen. Außerdem ist es möglich, verschiedene Klassen bzw. Kategorien zu erstellen, womit das Modell lernt, diese zu unterscheiden. Das Modell lernt in Echtzeit. Nach dem hochladen und anschließendem trainieren des Modells kann es getestet werden, um zu sehen, wie gut es neue Daten klassifizieren kann.

Wofür wird eine Teachable Machine verwendet?

Es gibt verschiedene Anwendungsbereiche für eine Teachable Machine. Es ist nicht nur möglich, Stimmen zu erkennen, sondern auch Audio und Bilder kategorisieren. Im Folgenden wird näher auf die verschiedenen Anwendungsbereiche eingegangen.

Die Anwendungsbereiche:

Bildklassifizierung

Damit ist die Erkennung und Kategorisierung von Bildern gemeint.
Beispiel: Mit dem Traing können Haustiere oder andere Objekte erkannt und klassifiziert werden.

Audioerkennung

Damit ist die Erkennung und Kategorisierung von Geräuschen oder Sprachbefehlen gemeint, welches auf Audio-Beispielen basiert.
Bespiel: Ein auf Klatschen oder eine spezifiche Stimme trainiertes Modell, welches auf diese reagiert.

Posenerkennung

Durch dieses Training können Körperposen für Interaktionen verfolgt oder Bewegungen analysiert werden. Diese Erkennung erfolgt dabei in Echtzeit.
Beispiel: Die Nutzung in Fitness-Apps zur Verfolgung von Übungsbewegungen oder in interaktiven Spielen, wie Just Dance.
